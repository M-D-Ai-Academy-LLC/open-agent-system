# Content Pipeline

A complete content creation pipeline that transforms rough notes into published articles.

## Quick Start

```bash
# Process raw notes into outline
/content process source/my-notes.md

# Write draft from outline
/content draft output-notes/my-notes-outline.md

# Edit the draft
/content edit output-drafts/my-notes-draft.md

# Publish final version
/content publish output-drafts/my-notes-edited.md

# Or run the full pipeline
/content pipeline source/my-notes.md
```
