# Document Summary - Example Stub

<!--
This is an example stub file showing how to request a document summary.

-->

## Source

- File: `source/example-document.pdf`

## Request

Summarize the key points of the document in a concise format suitable for
executive review. Focus on:
- Main findings or conclusions
- Action items or recommendations
- Critical data points

## Output

- **Format**: Markdown
- **Length**: 1 page (~300 words)
- **Audience**: Executive / Non-technical
- **Filename**: example-document-summary.md

## Notes

- Use bullet points for easy scanning
- Highlight any urgent items at the top
- Include a "Next Steps" section if applicable
